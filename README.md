# Flashcard

Your flashcard app is running! Here's what it does:

Home screen — lists your saved decks with due card counts
Create screen — describe a topic or paste text, then hit "Generate flashcards" to call Claude and build a deck
Study screen — SM-2 spaced repetition with flip animation; keyboard shortcuts (↑↓ to flip, 1/2/3 to grade)
Stats screen — review history, mastery progress, and per-deck breakdowns
Persistent storage — decks and progress are saved across sessions
